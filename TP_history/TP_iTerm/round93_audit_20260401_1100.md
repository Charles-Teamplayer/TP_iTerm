## 변경 사항

### scripts/session-registry.sh (200줄 변경, +99 -129)
- **fcntl.flock 파일 락 제거**: register/unregister 양쪽 모두에서 `REGISTRY_LOCK` + `fcntl.flock(LOCK_EX)` 삭제
  - `import fcntl` 제거, lock_path 환경변수 전달 제거
  - atomic write(mkstemp + os.replace)는 유지 → 파일 손상 방지는 여전히 동작
- **except Exception → bare except**: register 블록의 예외 처리 변경
- **REGISTRY_LOCK 변수 선언 제거** (스크립트 상단)

### scripts/stop-session.sh (28줄 변경, +2 -27)
- **--clear 명령 간소화**: Python3 임시파일+원자적 쓰기 19줄 → `echo JSON > file` 1줄로 대체
  - 기존: tempfile.mkstemp → os.rename → shutil.move(EXDEV fallback)
  - 변경: 단순 echo redirect (`>`)
- **get_valid_windows stderr 처리**: `2>/dev/null` 분리 → `2>&1` 통합
- **is_valid_window**: `2>/dev/null` 제거 (get_valid_windows에서 이미 처리)

## 주요 결정
- **fcntl 락 제거 근거**: 단일 사용자 macOS 환경에서 동시 쓰기 경합 가능성 낮음. atomic write(mkstemp+replace)로 충분.
- **--clear 간소화 근거**: stops.json은 초기화 전용이므로 cross-device fallback 불필요. 단순 redirect가 적절.
- **bare except 주의**: `except:` → `except Exception:` 이 아닌 bare except 사용 — KeyboardInterrupt 등도 잡힘. 후속 검토 대상.

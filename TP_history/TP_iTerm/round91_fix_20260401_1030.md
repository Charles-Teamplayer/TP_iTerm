### scripts/watchdog.sh
- 로그 로테이션: 줄 수(20000줄) 외에 파일 크기(300KB) 기준 추가
- **좀비 정리 로직 제거**: 감지(log)만 하고 자동 kill -9 제거 (오탐 방지)
  - 이전: 좀비 감지 → 부모/자식 포함 kill -9 → waitpid 유도
  - 이후: 좀비 감지 → 로그 기록만 (수동 판단 유도)
  - 좀비 awk 출력에 $4(command) 추가로 어떤 프로세스인지 식별 가능
- 중복실행 방지 lock: `set -C` (noclobber) 원자적 쓰기 → 단순 `-f` 파일 존재 체크로 변경
- SEC-002/SEC-004 보안 주석 및 chmod 0600 제거 (간소화)
- watchdog 루프 sleep: 30초 → 15초로 단축

## 주요 결정
- 좀비 프로세스 자동 kill 제거: 오탐으로 정상 프로세스가 죽는 사고 방지
- 로그 로테이션 이중 기준: 긴 줄이 적은 파일도 크기 기반 로테이션 가능
- 보안 강화(noclobber, chmod) 간소화: /tmp 파일에 과도한 보안 불필요 판단
